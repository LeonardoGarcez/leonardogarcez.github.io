BRANDVIZ

Um visual do PowerBi para auxiliar na visualização de projeções


Este é o Brandviz, um visual customizado para auxiliar os analistas a montarem gráficos que mesclam intervalos de confiança e linhas medidas e projetadas. O visual possui algumas edições de tamanhos e espessuras, permitindo ao usuário a alterar as características para proporcionar a melhor visualização dos dados.


Visualize seus dados de Forecast

O Brandviz está pronto para ilustrar seus dados da uma projeção com um ou dois intervalos de confiança.

Analise os Intervalos de Controle

Exiba gráficos de até duas faixas contemplando vários tipos de modelos de observação de medidas dentro de intervalos, como por exemplo o perfil de cartas de controle.
